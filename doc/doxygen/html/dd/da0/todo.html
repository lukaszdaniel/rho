Class rho::Subscripting

At present this class does not handle the special case (described in sec. 3.4.2 of the R Language Definition) where subscripting is being applied to an array, and there is a single subscript which is a matrix with as many columns as the array has dimensions, each row of the matrix being used to pick out a single element of the array.

A matter for possible review is the fact that during subsetting and subassignment operations, rho sometimes makes deep copies of vectors of GCEdge objects in cases where CR gets away with simple pointer copies.

CR 3 uses vectors of real numbers to represent subscripts where these would be too big to fit into an INTEGER.
