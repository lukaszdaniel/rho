Conduct a const visitor to protected objects in a specified range.

Conduct a GCNode::const_visitor object to each node pointed to by a pointer in the given range on the stack. Note that because rho does conservative stack scanning, it is possible that this may also visit some unreferenced objects.

Parameters
vPointer to the const_visitor object.
startPointer to the location on the stack to start at. If set nullptr, then starts at the base of the stack.
endPointer to the location on the stack to end at.

Note that as usual in C++, start and end form a semi-closed interval [start, end). In particular, if there is a pointer at end, it will not be visited.
