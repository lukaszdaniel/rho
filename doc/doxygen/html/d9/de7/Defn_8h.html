Detailed Description

A ragbag.

As rho development proceeds, the type definitions, many function prototypes etc. defined in this header file will disappear, because the relevant functionality will have been absorbed into the rho core, and declared within the appropriate header file in the src/include/rho directory.

In a few cases, a declaration within this file is repeated in a header file under src/include/rho; this is because source files within the rho core never #includes Defn.h itself (nor Rinternals.h. In such a case the relevant rho header file is #included back into Defn.h, so that the compiler can detect any inconsistency between the two declarations.
