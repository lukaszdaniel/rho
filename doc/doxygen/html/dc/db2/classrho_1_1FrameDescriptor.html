Detailed Description

A FrameDescriptor creates a static mapping between the symbols expected to be used in a function and integers that can be used as array offsets.

This is used to create the layout for Environment objects.

Note that it is not guaranteed that all symbols used in the function will be listed in the FrameDescriptor.
