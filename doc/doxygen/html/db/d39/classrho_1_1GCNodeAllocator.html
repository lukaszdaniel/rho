Detailed Description

An allocator for allocating and iterating over GCNode objects of different sizes.

This allocator should only be used to allocate GCNode objects, otherwise the garbage collector to be tricked into treating non-GCNodes as GCNodes during the Sweep phase of garbage collection.

rho::GCNode * rho::GCNodeAllocator::lookupPointer ( void *  candidate)
static

Find heap allocation start pointer.

This function finds the corresponding allocation for an internal or exact object pointer.

If the pointer is inside the bounds of the small object arena, then the corresponding superblock is found via pointer manipulation.

If the pointer is inside the heap bounds, then we iterate over all hash collisions for the pointer to find the corresponding allocation.
