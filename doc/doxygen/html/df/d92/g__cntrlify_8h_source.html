17 /* This header file is #include'd by g_cntrlify.c. It is a database rather
18  than a true header file: it includes arrays of escape sequences,
19  ligatures, etc., which govern the interpretation of a user-specified
20  text string, or `label'.
21 
22  This database is used in `controlification': the mapping of a label (a
23  string of unsigned chars) to a string of unsigned shorts, for further
24  processing. This mapping accomplishes two things: first, it maps
25  certain escape sequences, beginning with a backslash, to non-ASCII
26  ISO-Latin-1 characters (e.g., \mu is mapped to `times'), and second, it
27  maps certain escape sequences to internal control codes (e.g. \sp is
28  mapped to `start superscript'). */
29
