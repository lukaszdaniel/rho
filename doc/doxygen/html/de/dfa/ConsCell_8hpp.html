Detailed Description

Class rho::ConsCell and associated C interface.

To facilitate inlining of various ConsCell member functions, this file also includes the definition of class rho::PairList.

This file includes C functions for examining and setting the CAR and TAG of a rho::ConsCell; functions for examining and setting the CDR, and other operations accessing the tail of the list, are to be found in PairList.hpp.
