Detailed Description

template<typename T>
class rho::Allocator< T >

STL-compatible allocator front-ending rho::MemoryBank.

This templated class enables container classes within the C++ standard library to allocate their memory via rho::MemoryBank. However, its calls to MemoryBank are configured so that they do not give rise to garbage collections: this is to avoid any reentrant calls to the code for C++ standard library containers.

The code below is adapted from an example in the book "The C++ Standard Library - A Tutorial and Reference" by Nicolai M. Josuttis, Addison-Wesley, 1999. Also see Item 10 of Meyers' 'Effective STL' for the arcana of STL allocators.
