88  // explicit GCEdge(T* target) is intentionally not defined here.
89  //
90  // This prevents object initializers of the form
91  // Foo::Foo() : m_edge(expression_that_might_gc()) { ... }
92  // In that case, the expression is executed while the edge is
93  // uninitialized. If it causes a garbage collection, the GC's mark
94  // routine will attempt to follow the uninitialized edge, causing
95  // errors.
96  // Object initializers should be written as:
97  // Foo::Foo() { m_edge = expression_that_might_gc(); ... }
98  // which properly initialized the edge prior to doing the allocation.
