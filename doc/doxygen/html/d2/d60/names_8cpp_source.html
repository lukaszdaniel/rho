51 /* Table of .Internal(.) and .Primitive(.) R functions
52  * ===== ========= ==========
53  *
54  * Each entry is a line with
55  *
56  * printname c-entry offset eval arity pp-kind precedence rightassoc
57  * --------- ------- ------ ---- ----- ------- ---------- ----------
58  *2 name cfun code eval arity gram.kind gram.precedence gram.rightassoc
59  *3 PRIMNAME PRIMFUN PRIMVAL [*] PRIMARITY PPINFO PPINFO PPINFO
60  *
61  * where "2" are the component names of the FUNTAB struct (Defn.h)
62  * and "3" are the accessor macros. [*]: PRIMPRINT(.) uses the eval component
63  *
64  * printname: The function name in R
65  *
66  * c-entry: The name of the corresponding C function,
67  * actually declared in ../include/Internal.h .
68  * Convention:
69  * - all start with "do_",
70  * - all return SEXP.
71  * - all have argument list
72  * (SEXP call, SEXP op, SEXP args, SEXP env)
73  *
74  * quick_entry: Alternative C function, where the args are passed in an array
75  * for efficiency.
76  * Signature is RObject*(*)(const Expression* call,
77  * const BuiltInFunction* op,
78  * Environment* env,
79  * int num_args,
80  * RObject** args,
81  * const PairList* tags)
82  *
83  * offset: the 'op' (offset pointer) above; used for C functions
84  * which deal with more than one R function...
85  *
86  * eval: = XYZ (three digits) --- where e.g. '1' means '001'
87  * X=1 says that we should force R_Visible off
88  * X=0 says that we should force R_Visible on
89  * X=2 says that we should switch R_Visible on but let the C
90  * code update this.
91  * Y=1 says that this is an internal function which must
92  * be accessed with a .Internal(.) call, any other value is
93  * accessible directly and printed in R as ".Primitive(..)".
94  * Z=1 says evaluate arguments before calling (BUILTINSXP) and
95  * Z=0 says don't evaluate (SPECIALSXP).
96  *
97  * arity: How many arguments are required/allowed; "-1" meaning ``any''
98  *
99  * pp-kind: Deparsing Info (-> PPkind in ../include/Defn.h )
100  *
101  * precedence: Operator precedence (-> PPprec in ../include/Defn.h )
102  *
103  * rightassoc: Right (1) or left (0) associative operator
104  *
105  */
