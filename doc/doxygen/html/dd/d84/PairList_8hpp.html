Detailed Description

C interface of class PairList.

To facilitate inlining of various ConsCell member functions, the definition of class rho::PairList itself is in ConsCell.h.

This file includes C functions for examining and setting the CDR of a rho::ConsCell, and other operations accessing the tail of the list; functions for examining and setting the CAR and TAG of a rho:ConsCell are to be found in ConsCell.h.
