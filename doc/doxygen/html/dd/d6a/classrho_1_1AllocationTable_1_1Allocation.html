Detailed Description

A hash bucket for the allocation table.

Allocations are stored in multiple Allocation entries in the allocation table, however only one representative entry needs to be created and passed to insert(). The constructors of this class make it easy to construct that representative Allocation.

The two lowest bits of the data pointer are flags indicating if the hash bucket contains a superblock (2) and whether this hash bucket is the first representing that allocation (1).
