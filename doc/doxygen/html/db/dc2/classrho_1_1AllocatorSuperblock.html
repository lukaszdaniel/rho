Detailed Description

Superblocks are used to allocate small and medium sized objects. The bitset tracks which blocks are currently allocated.

The size of each block in the superblock is determined by m_size_class. The function bytesFromSizeClass(size_class) can be used to convert from size class to size in bytes. The size of the superblock itself is also determined by the size class, and is retrieved using superblockSize().

The index to the next untouched (never used) node in the superblock is stored in m_next_untouched. When m_next_untouched is equal to the number of blocks that fit in the superblock, then the superblock is taken out of the pool of superblocks with available nodes. Superblock nodes that are freed are linked up in freelists according to size class.
